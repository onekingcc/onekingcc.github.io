前言

感觉JavaScript里面的有些东西有趣但是有时候又有点不理解，有时候只是简单的用一下今天看了一下手册发现自己理解的还是有些偏差所以将自己所理解的记录下来！

什么是作用域

我理解的就是在一定区域内起作用的区域。

js的作用域是全局的，在代码的任何地方都有定义，都能使用
